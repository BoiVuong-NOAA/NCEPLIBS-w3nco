# W3NCO

Several utilities

Code manager: Boi Vuong


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang 


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-w3nco
make -j2
make install
```
